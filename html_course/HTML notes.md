## 01_Hello HTML

```html
<!DOCTYPE html>
<html lang="en">
<head>
	<meta charset="UTF-8">
	<title>My First Web Page</title>
	<style>
		html {
		font-size: 22px;
		}
	body {
		background-color: #333;
		color: whitesmoke;
		}
	</style>
</head>
<body>
	<h1>Hello World!</h1>
	<p>This is my first web page.</p>
</body>
</html>

```
